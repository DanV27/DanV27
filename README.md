## Featured Project: LearnPy

[LearnPy](https://github.com/DanV27/LearnPy) is a full-stack Python learning platform I built from scratch. It combines a hand-authored lesson catalog with in-browser code execution and an AI tutor.

**What it does:**
- 55+ lesson pages across 14 topics, each with a runnable coding challenge
- Challenges execute directly in the browser via [Pyodide](https://pyodide.org/) — no server-side code execution
- AI tutor (Claude API) generates custom lessons for topics not in the catalog and answers follow-up questions inline
- User auth with Flask-Login and password hashing via Werkzeug

**Stack:** Python, Flask, Flask-SQLAlchemy, Anthropic Claude API, Pyodide, Jinja2, Tailwind CSS

**Live demo:** [ai-code-generator--danv27.replit.app](https://learn-py--danv27.replit.app)
